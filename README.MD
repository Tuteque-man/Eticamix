# Eticamix

Eticamix is a web project built with HTML, CSS, and JavaScript, inspired by an interview with an IT professional. The project combines technical implementation with ethical reflection, presenting insights from the interview in a modern, accessible, and visually engaging format.

Live Demo: https://eticamix.netlify.app/

---

## Project Overview

- Purpose: Showcase front-end development skills while exploring ethical issues in technology.  
- Content: Based on an interview with an IT professional, covering topics such as:
  - The role of ethics in technology
  - Responsibilities of IT professionals
  - The social impact of digital innovation  
- Technologies:
  - HTML5 — semantic structure
  - CSS3 — responsive design, gradients, animations
  - JavaScript (ES6) — interactivity and dynamic features

---

## Features

- Responsive layout for desktop and mobile  
- Gradient text effects and modern UI styling  
- Interactive sections powered by JavaScript  
- Clean and accessible design principles
